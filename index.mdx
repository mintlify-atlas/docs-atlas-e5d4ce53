## How it works

GBrain is a thin retrieval layer that sits between your markdown brain repo and your AI agent. You keep writing markdown files — GBrain makes them searchable.

<Steps>
  <Step title="Import your markdown">
    Point GBrain at any directory of markdown files. It chunks, embeds, and indexes everything — idempotently, using SHA-256 content hashing to skip unchanged files.
    ```bash
    gbrain import ~/git/brain/
    ```
  </Step>
  <Step title="Search with your agent">
    Your AI agent calls `gbrain query` or the MCP `query` tool. GBrain runs hybrid search — keyword + vector + multi-query expansion — and returns ranked results with stale-page alerts.
    ```bash
    gbrain query "what do we know about competitive dynamics?"
    ```
  </Step>
  <Step title="Keep the brain alive">
    `gbrain sync --watch` polls your git repo and imports only what changed. The agent enriches pages after meetings, ingests transcripts, and adds timeline entries. The brain compounds over time.
    ```bash
    gbrain sync --watch --repo ~/git/brain/
    ```
  </Step>
</Steps>
